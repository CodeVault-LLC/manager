# Security Policy

Reported vulnerabilities can be found on [cve.mitre.org](https://cve.mitre.org/cgi-bin/cvekey.cgi?keyword=manager).

## Reporting a Vulnerability

We want to keep Manager safe for everyone.
If you've discovered a security vulnerability in Manager,
we appreciate your help in disclosing it to us in a responsible manner,
by creating a [security advisory](https://github.com/codevault-llc/manager/security/advisories).
